Hello There !

To run this Music Bot you need follow this easy guide!

Go to https://discord.com/developers/applications (Here you need use the same Discord Account and the same ID for some commands in file)

1 - Create New App 

![image](https://user-images.githubusercontent.com/119908735/219932129-ac94ca3b-ea2f-4a53-9e32-a0ff86ac1d29.png)

2 - Choose a name 

![image](https://user-images.githubusercontent.com/119908735/219932155-dfc078ba-58c6-4190-9558-12a7e40ede46.png)

3 - Create New Bot 

![image](https://user-images.githubusercontent.com/119908735/219932176-ea0878aa-d815-4665-ab94-90f96aa8bcab.png)

After this you need save your Bot Token !

![image](https://user-images.githubusercontent.com/119908735/219932222-28f30418-690e-4bcc-88e9-f3bdae451995.png)
![image](https://user-images.githubusercontent.com/119908735/219932272-26c684ae-3a80-46bc-8473-1868f719361c.png)

4 - Giving permissions and add to a Discord Server : 

![image](https://user-images.githubusercontent.com/119908735/219932554-fdbaa1b3-144a-4688-b1d4-bf6953de9d15.png)
![image](https://user-images.githubusercontent.com/119908735/219932562-f4b6c489-f3d2-43bd-aa99-8c775fe2bd41.png)

Go to OAuth2 and Select URL Generator (For now just give the Admin permissions and Bot Scope, you can change this later if you needed)

Now you have this link : 

![image](https://user-images.githubusercontent.com/119908735/219932652-857fca6c-28fb-49fa-8e8c-f6e22d4f62f5.png)

Just copy and paste in your browser : 

![image](https://user-images.githubusercontent.com/119908735/219932711-aba5f94e-8313-4469-a299-cedd2e56204c.png)

Click on Authorize and you can see you Bot in your server!


Now go to your Discord app or web and Log In.
In settings :

Go to Advanced and Turn on Dev. Mode 

![image](https://user-images.githubusercontent.com/119908735/219932333-2c24b708-88f0-4ee3-aa9c-8338be62bc90.png)

Now right click on your profile in any server you have and click Copy ID

![image](https://user-images.githubusercontent.com/119908735/219932377-58a11e69-3a15-4715-b6e9-c214aa08afc7.png)
Keep this ID with your Bot Token and go to despr0grammer-music-bot.py

First go to line 872 and paste your Bot Token there, INSIDE 

![image](https://user-images.githubusercontent.com/119908735/219932429-89718f1d-5bc1-4308-bd9e-7ab4cf074f4f.png)

Now in lines : 844 and 199 paste your Discord ID and run the file!
